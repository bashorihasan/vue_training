01 - Getting Started with Vue.js

Vue.js is a progressive JavaScript framework for building user interfaces. Let's start with the basics: data binding and interpolation.

Example 1: Simple Text Interpolation

Vue uses double curly braces {{ }} for text interpolation:

{{ message }}

Hello, {{ name }}!

<div id="app1">
  <p>{{ message }}</p>
  <p>Hello, {{ name }}!</p>
</div>

const app1 = Vue.createApp({
  data() {
    return {
      message: 'Welcome to Vue.js!',
      name: 'Developer'
    }
  }
}).mount('#app1');

Example 2: Expressions in Templates

You can use JavaScript expressions inside the curly braces:

Number: {{ number }}

Double: {{ number * 2 }}

Message reversed: {{ message.split('').reverse().join('') }}

Is positive: {{ number > 0 ? 'Yes' : 'No' }}

<div id="app2">
  <p>Number: {{ number }}</p>
  <p>Double: {{ number * 2 }}</p>
  <p>Message reversed: {{ message.split('').reverse().join('') }}</p>
  <p>Is positive: {{ number > 0 ? 'Yes' : 'No' }}</p>
</div>

Example 3: Attribute Binding with v-bind

Use v-bind (or shorthand :) to bind attributes:

{{ linkText }}

Hover over me to see a tooltip!

<img v-bind:src="imageUrl" v-bind:alt="imageAlt">
<a :href="link">{{ linkText }}</a>
<p v-bind:title="tooltip">Hover over me!</p>

Example 4: Dynamic Styling

Bind styles and classes dynamically:

Styled text with color: {{ textColor }}
Box with multiple styles 
<div :style="{ color: textColor, fontSize: fontSize + 'px' }">
  Styled text
</div>
<div :style="boxStyle">
  Box with multiple styles
</div>
