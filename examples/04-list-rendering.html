04 - List Rendering

The v-for directive is used to render lists based on arrays and objects. Always use a unique :key for better performance and to avoid rendering issues.

Example 1: Basic Array Rendering

Render an array of items:

Programming Languages:

  • {{ index + 1 }}. {{ language }} 
<ul>
  <li v-for="(language, index) in languages" :key="index">
    {{ index + 1 }}. {{ language }}
  </li>
</ul>

data() {
  return {
    languages: ['JavaScript', 'Python', 'Java', 'C++', 'Ruby']
  }
}

Example 2: Array of Objects

Render complex data structures:

Task List:

{{ task.title }} ✅ ⏳
{{ task.completed ? 'Done' : 'Pending' }} 
<div v-for="task in tasks" :key="task.id" class="list-item">
  <strong>{{ task.title }}</strong>
  <span v-if="task.completed"> ✅</span>
</div>

data() {
  return {
    tasks: [
      { id: 1, title: 'Learn Vue.js', completed: true },
      { id: 2, title: 'Build a project', completed: false }
    ]
  }
}

Example 3: Object Iteration

Iterate over object properties:

User Information:

{{ key }}: {{ value }} 
<div v-for="(value, key) in user" :key="key">
  <strong>{{ key }}:</strong> {{ value }}
</div>

data() {
  return {
    user: {
      name: 'John Doe',
      email: 'john@example.com',
      role: 'Developer',
      location: 'New York'
    }
  }
}

Example 4: Dynamic List Management

Add and remove items from a list:

{{ item.text }}

No items yet. Add some!

<input v-model="newItem" @keyup.enter="addItem">
<button @click="addItem">Add Item</button>

<div v-for="(item, index) in items" :key="item.id">
  {{ item.text }}
  <button @click="removeItem(index)">Remove</button>
</div>

methods: {
  addItem() {
    if (this.newItem.trim()) {
      this.items.push({
        id: Date.now(),
        text: this.newItem
      });
      this.newItem = '';
    }
  },
  removeItem(index) {
    this.items.splice(index, 1);
  }
}

Example 5: Nested Lists

Render nested arrays:

{{ category.name }}

  • {{ item }} 
<div v-for="category in categories" :key="category.name">
  <h4>{{ category.name }}</h4>
  <ul>
    <li v-for="item in category.items" :key="item">
      {{ item }}
    </li>
  </ul>
</div>
