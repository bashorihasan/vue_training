05 - Two-Way Binding

The v-model directive creates two-way data binding on form inputs, textarea, and select elements. It automatically updates the data when the user inputs something.

Example 1: Text Input

Bind text inputs with v-model:

Live Preview: {{ name }}

Length: {{ name.length }} characters

<input v-model="name" placeholder="Type your name">
<p>{{ name }}</p>

data() {
  return {
    name: ''
  }
}

Example 2: Textarea

Two-way binding with textarea:

Message Preview:

{{ message }}

Word Count: {{ wordCount }}

<textarea v-model="message"></textarea>
<p>{{ message }}</p>

computed: {
  wordCount() {
    return this.message.trim() ? this.message.trim().split(/\s+/).length : 0;
  }
}

Example 3: Checkboxes

Single and multiple checkbox binding:

Agreement status: {{ agree ? 'Agreed' : 'Not agreed' }}

Selected: {{ hobbies.join(', ') || 'None' }}

<input type="checkbox" v-model="agree">
<input type="checkbox" v-model="hobbies" value="Reading">
<input type="checkbox" v-model="hobbies" value="Gaming">

data() {
  return {
    agree: false,
    hobbies: []
  }
}

Example 4: Radio Buttons

Radio button binding:

Selected: {{ framework || 'None' }}

<input type="radio" v-model="framework" value="Vue">
<input type="radio" v-model="framework" value="React">
<input type="radio" v-model="framework" value="Angular">

data() {
  return {
    framework: 'Vue'
  }
}

Example 5: Select Dropdown

Dropdown select binding:

Selected Country: {{ country || 'None' }}

Selected Skills: {{ skills.join(', ') || 'None' }}

<select v-model="country">
  <option value="USA">United States</option>
  <option value="UK">United Kingdom</option>
</select>

<select v-model="skills" multiple>
  <option value="HTML">HTML</option>
  <option value="CSS">CSS</option>
</select>

Example 6: Modifiers

v-model modifiers: .lazy, .number, .trim:

Value: "{{ normal }}"

Value: "{{ lazy }}"

Value: {{ age }} (Type: {{ typeof age }})

Value: "{{ trimmed }}" (Length: {{ trimmed.length }})

<input v-model="normal">
<input v-model.lazy="lazy">
<input v-model.number="age" type="number">
<input v-model.trim="trimmed">
