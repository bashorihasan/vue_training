02 - Event Handling

Vue provides the v-on directive (shorthand: @) to listen to DOM events and execute methods when they're triggered.

Example 1: Basic Click Event

Click the button to increment a counter:

{{ count }}
<div class="counter">{{ count }}</div>
<button v-on:click="increment">Increment</button>
<button @click="decrement">Decrement</button>
<button @click="reset">Reset</button>

methods: {
  increment() {
    this.count++;
  },
  decrement() {
    this.count--;
  },
  reset() {
    this.count = 0;
  }
}

Example 2: Event with Parameters

Pass parameters to event handlers:

{{ message }}

<button @click="greet('Hello')">Say Hello</button>

methods: {
  greet(greeting) {
    this.message = `${greeting}, Vue.js Developer!`;
  }
}

Example 3: Accessing Event Object

Access the native event object:

{{ info }}

<button @click="handleEvent">Click Me</button>
<input @keyup="handleKeyup">

methods: {
  handleEvent(event) {
    this.info = `Button clicked at: ${event.clientX}, ${event.clientY}`;
  },
  handleKeyup(event) {
    this.info = `Key pressed: ${event.key}`;
  }
}

Example 4: Event Modifiers

Use event modifiers like .prevent, .stop, .once:

{{ submissionMessage }}


{{ onceMessage }}

<form @submit.prevent="handleSubmit">
  <input v-model="formInput">
  <button type="submit">Submit</button>
</form>

<button @click.once="showOnceMessage">Click Once Only</button>

methods: {
  handleSubmit() {
    this.submissionMessage = `Form submitted: ${this.formInput}`;
  }
}

Example 5: Key Modifiers

Listen for specific keys:

{{ keyMessage }}

<input @keyup.enter="onEnter" placeholder="Press Enter">
<input @keyup.esc="onEscape" placeholder="Press Escape">
<input @keyup.space="onSpace" placeholder="Press Space">

methods: {
  onEnter() {
    this.keyMessage = 'Enter key pressed!';
  },
  onEscape() {
    this.keyMessage = 'Escape key pressed!';
  },
  onSpace() {
    this.keyMessage = 'Space key pressed!';
  }
}
