07 - Watchers

Watchers allow you to perform asynchronous operations or expensive operations in response to changing data. Use them when you need to perform actions when data changes.

Example 1: Basic Watcher

Watch for changes in data:

You typed: {{ question }}

Character count: {{ characterCount }}

{{ log }} 
watch: {
  question(newValue, oldValue) {
    this.logs.push(
      `Changed from "${oldValue}" to "${newValue}"`
    );
  }
}

Example 2: Async Operation

Perform async operations when data changes:

üîç Searching...
Found results for: "{{ searchResult }}"

Simulates API call with 1 second delay

watch: {
  searchTerm(newTerm) {
    this.isSearching = true;
    this.searchResult = '';
    
    // Simulate API call
    setTimeout(() => {
      this.searchResult = newTerm;
      this.isSearching = false;
    }, 1000);
  }
}

Example 3: Deep Watcher

Watch nested object properties with deep option:

User Profile:





Last Change: {{ lastChange }}

Change Count: {{ changeCount }}

watch: {
  user: {
    handler(newValue) {
      this.lastChange = new Date().toLocaleTimeString();
      this.changeCount++;
    },
    deep: true  // Watch nested properties
  }
}

Example 4: Immediate Watcher

Run watcher immediately on component creation:

{{ validationMessage }} 
watch: {
  username: {
    handler(newValue) {
      if (!newValue) {
        this.validationMessage = 'Username is required';
        this.isValid = false;
      } else if (newValue.length < 3) {
        this.validationMessage = 'Username must be at least 3 characters';
        this.isValid = false;
      } else {
        this.validationMessage = 'Username is valid';
        this.isValid = true;
      }
    },
    immediate: true  // Run on initialization
  }
}

Example 5: Multiple Watchers

Watch multiple properties and combine results:



Sum: {{ sum }}

Product: {{ product }}

watch: {
  num1(newVal) {
    this.updateCalculations();
  },
  num2(newVal) {
    this.updateCalculations();
  }
},
methods: {
  updateCalculations() {
    this.sum = this.num1 + this.num2;
    this.product = this.num1 * this.num2;
    this.calculations.push(
      `${this.num1} + ${this.num2} = ${this.sum}, ` +
      `${this.num1} √ó ${this.num2} = ${this.product}`
    );
  }
