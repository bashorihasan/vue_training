03 - Conditional Rendering

Vue provides directives like v-if, v-else-if, v-else, and v-show to conditionally render elements based on data.

Example 1: v-if, v-else-if, v-else

Use v-if for conditional rendering. Elements are added/removed from DOM:

‚úÖ This text is visible!

‚ùå This text is hidden!

<button @click="toggleVisibility">Toggle</button>
<p v-if="isVisible">‚úÖ This text is visible!</p>
<p v-else>‚ùå This text is hidden!</p>

data() {
  return {
    isVisible: true
  }
},
methods: {
  toggleVisibility() {
    this.isVisible = !this.isVisible;
  }
}

Example 3: v-show

v-show toggles CSS display property (element stays in DOM):

This uses v-show (inspect to see it's always in DOM)

Note: v-show is better for frequent toggles, v-if for less frequent changes.

<button @click="showMessage = !showMessage">Toggle</button>
<p v-show="showMessage">This uses v-show</p>

// v-show toggles display: none
// Element stays in DOM, just hidden
// Better for frequent toggles

Example 4: Conditional Rendering with Template

Group multiple elements with <template>:

<template v-if="isLoggedIn">
  <h4>Welcome back!</h4>
  <p>You are logged in.</p>
  <p>Access granted.</p>
</template>

<template v-else>
  <h4>Please log in</h4>
  <p>You need to log in.</p>
</template>
