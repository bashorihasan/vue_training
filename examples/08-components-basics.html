08 - Components Basics

Components are reusable Vue instances with a name. They accept the same options as the root Vue instance, such as data, computed, watch, methods, and lifecycle hooks.

Example 1: Simple Component

Create a basic reusable component:

Each component maintains its own state!

app.component('button-counter', {
  data() {
    return {
      count: 0
    }
  },
  template: `
    <button @click="count++">
      Clicked {{ count }} times
    </button>
  `
});

Example 2: Component with Template

Components with more complex templates:

This is an informational alert box component. Your action was completed successfully. Please review this important message. 
app.component('alert-box', {
  props: ['type', 'title'],
  template: `
    <div :class="'alert-box alert-' + type">
      <h4>{{ title }}</h4>
      <slot></slot>
    </div>
  `
});

Example 3: Component with Methods

Components can have their own methods:

app.component('greeting-card', {
  data() {
    return {
      name: '',
      greetings: []
    }
  },
  methods: {
    addGreeting() {
      if (this.name.trim()) {
        this.greetings.push(this.name);
        this.name = '';
      }
    }
  },
  template: `
    <div class="card">
      <h4>Greeting Card</h4>
      <input v-model="name" @keyup.enter="addGreeting">
      <button @click="addGreeting">Add</button>
      <p v-for="(greeting, index) in greetings" :key="index">
        Hello, {{ greeting }}! üëã
      </p>
    </div>
  `
});

Example 4: Component Lifecycle

Components have lifecycle hooks:

app.component('lifecycle-demo', {
  data() {
    return {
      message: 'Component is mounted!'
    }
  },
  mounted() {
    console.log('Component mounted!');
  },
  unmounted() {
    console.log('Component unmounted!');
  },
  template: `
    <div class="card">
      <h4>{{ message }}</h4>
      <p>Check the console for lifecycle logs</p>
    </div>
  `
});

Example 5: Slot Content

Use slots to pass content to components:

app.component('feature-card', {
  template: `
    <div class="card">
      <slot name="header"></slot>
      <slot></slot>
      <slot name="footer"></slot>
    </div>
  `
});
