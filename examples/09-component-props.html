09 - Component Props

Props are custom attributes you can register on a component. They allow you to pass data from a parent component to a child component.

Example 1: Basic Props

Pass data to child components using props:

app.component('blog-post', {
  props: ['title', 'author'],
  template: `
    <div class="post">
      <h4>{{ title }}</h4>
      <p class="post-meta">By {{ author }}</p>
    </div>
  `
});

Example 2: Props with Types

Define prop types for validation:

app.component('user-profile', {
  props: {
    name: String,
    age: Number,
    email: String
  },
  template: `
    <div class="card user-card">
      <div class="avatar">{{ name[0] }}</div>
      <div class="user-info">
        <h4>{{ name }}</h4>
        <p>Age: {{ age }} | Email: {{ email }}</p>
      </div>
    </div>
  `
});

Example 3: Props with Default Values

Set default values for props:

app.component('product-card', {
  props: {
    name: {
      type: String,
      required: true
    },
    price: {
      type: Number,
      default: 0
    },
    inStock: {
      type: Boolean,
      default: true
    }
  },
  template: `
    <div class="product-card">
      <h4>{{ name }}</h4>
      <p class="product-price">\${{ price.toFixed(2) }}</p>
      <span :class="inStock ? 'product-stock in-stock' : 'product-stock out-of-stock'">
        {{ inStock ? 'In Stock' : 'Out of Stock' }}
      </span>
    </div>
  `
});

Example 4: Object Props

Pass objects as props:

app.component('article-card', {
  props: {
    article: {
      type: Object,
      required: true
    }
  },
  template: `
    <div class="card">
      <h4>{{ article.title }}</h4>
      <p>{{ article.content }}</p>
      <div>
        <span v-for="tag in article.tags" 
              :key="tag" 
              class="badge"
              style="background: #42b983; color: white;">
          {{ tag }}
        </span>
      </div>
      <p class="post-meta">Published: {{ article.date }}</p>
    </div>
  `
});

Example 5: Prop Validation

Add custom validation to props:

app.component('rating-display', {
  props: {
    rating: {
      type: Number,
      required: true,
      validator(value) {
        return value >= 0 && value <= 5;
      }
    }
  },
  computed: {
    stars() {
      return '⭐'.repeat(Math.floor(this.rating));
    }
  },
  template: `
    <div class="card">
      <div>{{ stars }}</div>
      <p>Rating: {{ rating }} / 5</p>
    </div>
  `
});
